# 3D Math Adventure / 3D 數學冒險樂園

這是一個給小朋友使用的雙語數學網站遊戲，提供 `K1`、`K2`、`K3` 和 `P1` 四個學習級別。

孩子可以在 3D 立方體、3D 風格水果圖示和小角色提示中練習計算，並且可以隨時在中文與英文介面之間切換。

## 功能

- 中英文即時切換
- K1、K2、K3、P1 分級題目
- 3D 視覺化數字立方體
- 3D 風格水果／星星圖示與小角色提示
- 內建簡單音效（答對、答錯、切換題目）
- 卡通冒險地圖與關卡路線
- 加法、減法、P1 簡單乘法
- 分數、連續答對、已完成題數追蹤
- 大型按鈕，適合小朋友操作

## 級別設計

- `K1`: 1 至 5 的簡單加法
- `K2`: 10 以內加減法
- `K3`: 20 以內加減法
- `P1`: 20 以內加減法及簡單乘法

## 使用方式

1. 直接用瀏覽器打開 `index.html`
2. 選擇 `K1`、`K2`、`K3` 或 `P1`
3. 如需要，可隨時切換 `中文 / English`
4. 按答案作答，答對後會自動出下一題
5. 觀察水果或星星圖示，幫助孩子用「看圖數量」方式理解題目
6. 留意左邊的冒險地圖與音效提示，增加遊戲感

## 簡單驗證

- 切換語言後，標題、按鈕、提示和級別說明會即時更新
- 題目切換時，水果／星星圖示和角色提示會跟著更新
- 切換級別時，冒險地圖會高亮目前級別
- 點擊答案或下一題時，會播放簡單提示音
- 答對時會加 `10` 分，並增加已完成題數
- 答錯時會顯示提示，但不會增加完成題數

## 適合用途

- 家長陪伴學習
- 幼稚園及初小數學練習
- 雙語數學入門互動網站示範
